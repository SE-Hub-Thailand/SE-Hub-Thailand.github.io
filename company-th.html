โครงการรับสมัครสมาชิกของ ATSI และรับสมัคร Sponsor สำหรับ Project ต่างๆ ของ ATSI

ATSI และ SE HUB นำเสนอโครงการพัฒนาศักยภาพดิจิทัล

แจก Course ฟรี บริษัทละ 1 course ต่อ 1 ที่นั่ง ใน 5 course ของเรา(มูลค่ารวม 60,000 บาท)
พิเศษ สำหรับสมาชิกสมาคม ATSI สามารถ สมัคร package premium เพียง 2,000 บาท เพื่อรับ course เพิ่ม อีกบริษัท ละ 1 ที่ ต่อ 1 course

เนื่องด้วย ATSI [สมาคมอุตสาหกรรมซอฟต์แวร์ไทย] ร่วมกับ SE HUB มีความประสงค์ที่จะนำเสนอโครงการ พัฒนาศักยภาพบุคลากรด้านการพัฒนาซอฟต์แวร์และทักษะความรู้ความชำนาญในการใช้เครื่องมือในยุคดิจิทัลแบบครอบคลุมทั่วประเทศ

Course ที่นำเสนอ:

  • 1. Open code lab camp - เรียน 4 เดือนเต็ม จบมาเป็น Programmer ได้
  • 2. Pre camp open code lab - เรียน 6 วันเต็ม มูลค่า 12,000 บาท (แจกฟรี จำนวนจำกัด)
  • 3. Digital transformation camp - เรียน 6 วันเต็ม มูลค่า 12,000 บาท (แจกฟรี จำนวนจำกัด)
  • 4. Low Code Camp - เรียน 6 วันเต็ม มูลค่า 12,000 บาท (แจกฟรี จำนวนจำกัด)
  • 5. TAs boot Camp - เรียน 6 วันเต็ม มูลค่า 12,000 บาท (แจกฟรี จำนวนจำกัด)
  • 6. English IT Camp - เรียน 6 วันเต็ม มูลค่า 12,000 บาท (แจกฟรี จำนวนจำกัด)

ซึ่งโครงการนี้รับผิดชอบดำเนินโครงการโดย คุณ กฤษฏา เฉลิมสุข (คุณโอ๊ค) ผู้ร่วมก่อตั้งและอดีตนายกสมาคมโปรแกรมเมอร์ไทย ซึ่งเป็นผู้มีประสบการณ์ การปั้น บุคลากร Programmer มาแล้วไม่ต่ำกว่า 300 ราย

แนวคิดของโครงการ:

แนวคิดหลักของโครงการคือ การพัฒนาบุคคลากรด้านเทคโนโลยีดิจิทัลที่มีคุณภาพเพื่อรองรับต่อความต้องการของผู้ประกอบการและประเทศ และ เพื่อส่งเสริมพัฒนาบุคลากรที่มีคุณภาพได้อย่างต่อเนื่องและยั่งยืน

เราเชื่อว่าโครงการนี้จะมีผลที่ดีต่อผู้ประกอบการและต่อสังคม ซึ่งเรากำลังมองหาผู้ร่วมสนับสนุนที่การดำเนินการโครงการได้อย่างต่อเนื่อง

ทั้งนี้ทางสมาคม มีความประสงค์ที่จะแจกทุนสำหรับหรับ ทุกๆ แคมป์ ให้กับทางผู้ประกอบการ ได้ส่งพนักงานเข้ามา upskill ฟรี 1 ท่าน ในทุกๆ camp (*ยกเว้น open code lab camp)

  • อยู่ข้างหน้าคู่แข่งด้วยการเข้าถึงเนื้อหาที่คัดเลือกของเรา
  • ได้รับสิทธิพิเศษและโปรโมชันเฉพาะสมาชิก
  • เป็นส่วนหนึ่งของชุมชนที่ให้ความสำคัญกับการแบ่งปันความรู้และนวัตกรรม

แพ็กเกจสมาชิกพรีเมียม

  • เข้าถึงหลักสูตรที่ทันสมัย 6 หลักสูตรต่อปี เพื่อเสริมสกิลของทีมของคุณ
  • โชว์เคสใน E-Catalog ของเรา 2 หน้า เพื่อเพิ่มความเห็นชอบต่อแบรนด์ของคุณ
  • รับคำปรึกษาดิจิทัลจากผู้เชี่ยวชาญ 1 ชั่วโมง เพื่อแนะนำกลยุทธ์ธุรกิจของคุณ
  • โอกาสที่เด่นเพื่อปรากฏบน "ช่องดิจิทัล" เพิ่มการเข้าถึง
  • รับส่วนลดพิเศษในการประชุมวิชาการด้านเทคโนโลยีและเข้าถึงงานจัดหางานของเราเป็นพิเศษ

การสนับสนุนเพื่อการสร้างแบรนด์

  • จัดตำแหน่งแบรนด์ของคุณในทุกวัสดุของระบบนิเวศของเรา
  • สร้างความประทับใจให้กับฐานผู้ชมของเรา
  • ได้รับการเปิดเผยและการยอมรับในกิจกรรมและเวบินาร์
  • รับการจัดวางที่มีลำดับความสำคัญในการเผยแพร่และคุณสมบัติของเรา
  • ใช้ประโยชน์จากโอกาสในการเครือข่ายกับพาร์ทเนอร์และผู้มีส่วนได้ส่วนเสียของเรา

การสนับสนุนเพื่อการรับสมัคร

  • สนับสนุนค่าเรียนของนักเรียนและได้รับพนักงานที่มีความทุ่มเทในการทำงานอย่างน้อยหนึ่งปี
  • รูปร่างกำลังแรงงานอนาคตด้วยการสนับสนุนการศึกษาของพวกเขา
  • รับความกตัญญูและความภักดีต่อแบรนด์จากนักเรียนที่ได้รับการสนับสนุน
  • เพิ่มชื่อเสียงของแบรนด์ของคุณในฐานะผู้สนับสนุนและที่ปรึกษาในอุตสาหกรรม
  • รับประโยชน์จากการได้รับส่วนลดภาษีและการเผยแพร่ข่าวที่มีผลต่อความนิยม

การสนับสนุนสำหรับงานจัดหางาน

  • การเผยแพร่แบรนด์อย่างเป็นเอกลักษณ์ในงานจัดหางานของเราทุกๆสองเดือน
  • เข้าถึงความสามารถที่ยังไม่ได้ถูกสัมผัสจากมหาวิทยาลัยหลายแห่ง
  • มีโอกาสพูดคุยโต้ตอบโดยตรงกับผู้ที่มีศักยภาพในการรับเข้าทำงานในบริษัท
  • จัดตำแหน่งบริษัทของคุณให้เป็นที่ต้องการของผู้สมัคร
  • เพิ่มความเห็นชอบต่อแบรนด์ในหมู่ผู้สำเร็จการศึกษาใหม่

การประชุมเทคโนโลยี

  • สัมผัสกับวิทยากรและผู้นำความคิดจากทั่วประเทศไทย
  • สร้างเครือข่ายกับผู้เข้าร่วมงานมากกว่า 300 คนและผู้ที่มีศักยภาพในการทำงานร่วมกันในทุกกิจกรรม
  • อัปเดตเทรนด์และนวัตกรรมเทคโนโลยีล่าสุด
  • มีโอกาสเพื่อการเผยแพร่แบรนด์ในกลุ่มผู้ฟังที่มีความรู้ทางเทคโนโลยี
  • เพิ่มการพัฒนาทางวิชาชีพและการเชื่อมโยงในอุตสาหกรรมสำหรับบริษัทของคุณ

E-แคตตาล็อก

  • นำเสนอผลิตภัณฑ์หรือบริการของคุณให้กับกลุ่มผู้ฟังที่กว้างใน e-แคตตาล็อกของเรา
  • ได้รับประโยชน์จากเครือข่ายการกระจายของเราที่มีบริษัท, รัฐบาล, และมหาวิทยาลัยมากกว่า 500 แห่ง
  • เพิ่มการรับรู้และการรู้จักแบรนด์ในแคตตาล็อกแบบพรีเมียม
  • มีตัวเลือกเพื่อเพิ่มการเผยแพร่แบรนด์ของคุณด้วยหน้าเพิ่มเติม
  • ขับเคลื่อนการเยี่ยมชมที่มุ่งมั่นและสร้างโอกาสธุรกิจจากผู้อ่านที่หลากหลาย

ช่องทางดิจิทัล

  • ได้รับการเผยแพร่อย่างมีนัยยะโดยการปรากฏในช่องทางดิจิทัลที่เราให้ความเห็นชอบ
  • มีส่วนร่วมกับฐานผู้ฟังของเรา เพิ่มความจำได้ของแบรนด์
  • ทำงานร่วมกับผู้มีอิทธิพลและผู้นำความคิดในวงการเทคโนโลยี
  • การผลิตวิดีโอที่ถูกปรับให้เหมาะสม แน่นอนว่าจะแทนแบรนด์ของคุณในรูปแบบมืออาชีพ
  • ได้รับประโยชน์จากการส่งเสริมข้ามแพลตฟอร์มบนเว็บไซต์, โซเชียลมีเดีย, และแพลตฟอร์มอื่นๆ

บริการปรึกษา

  • ได้รับประโยชน์จากคำแนะนำของเราที่ปรับเปลี่ยนให้เหมาะกับความต้องการธุรกิจเฉพาะของคุณ
  • รับกลยุทธ์ที่สามารถดำเนินการเพื่อขับเคลื่อนการนำเข้าดิจิทัลและปรับปรุงการประสิทธิภาพ
  • การประเมินที่จะเกิดขึ้นประจำปีเพื่อตรวจสอบความก้าวหน้าและการปรับปรุง
  • การบริการที่รวดเร็ว, มืออาชีพ และเป็นมิตร
  • รับประโยชน์จากการมีผู้ปรึกษาที่มีความรู้และประสบการณ์ในอุตสาหกรรม
